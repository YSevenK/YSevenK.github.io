搭建个人博客

本文最后更新于：2023年9月30日 下午

第一篇博客–搭建个人博客。参考B站大佬UP主程序羊的视频->[点击直达](【手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo】 https://www.bilibili.com/video/BV1Yb411a7ty/?share_source=copy_web&vd_source=7410be56f3d16987da62fe3d774e9a67)

一、环境介绍

Windows10,nodejs:16,hexo-cli:4.3.1,git

二、环境准备

1.最好安装好git。

2.安装nodejs，直接百度搜就可以，或者[点击这里](Node.js (nodejs.org))，建议不要下载最新版，(另外插一嘴，nodejs有一个版本管理工具，需要的可以自行百度一下)，nodejs直接傻瓜式安装就可以，安装完成后win+R输入cmd，输入命令：node -v显示版本号即为安装成功

三、安装Heox

1.在安装hexo之前可以安装一下淘宝源cnpm，速度比较快（我用的npm，所以后面的步骤都使用的npm，我也没有用过cnpm，不太清楚具体的操作，大家自行选择），打开终端输入命令：npm install -g cnpm --registry=https://registry.npm.taobao.org命令安装，安装好之后输入cnpm显示全部的工具，输入cnpm -v查看版本。

2.输入命令：npm install -g hexo-cli 来安装hexo框架，输入命令：hexo -v查看版本。

出现版本号，表示安装成功。

四、搭建博客

创建一个文件夹，之后博客的内容都会在此文件夹中生成，可以在终端中输入命令：mkdir 文件夹名，或者直接打开文件管理器新建-简单粗暴。

在终端中进入到刚刚创建的文件夹中输入命令：hexo init初始化博客，可能时间有点长，完成后会提示**Start blogging with Hexo !**，输入命令：dir可以查看生成了哪些文件。

输入命令hexo -s即可启动博客，会给出一个本地的地址，打开就可以看到我们的博客了，默认自动生成一篇博客，命令：ctrl+c停止运行

输入命令：hexo -n 博客名新建博客，新建的博客会在\source\_posts目录下，可以使用vim编辑，也可以用其他支持makedown语法的软件打开。编辑完成后回到终端，输入命令:hexo -clean清除缓存，输入命令：hexo -g生成文件，在重新启动一下，就可以看到我们刚刚新建的博客了。

五、部署

我们的博客部署在GitHub上，点开GitHub，新建仓库，部署个人博客仓库的名称一定要符合要求，一定是 你的用户名.github.io，描述加不加都可，其他都默认，点击创建。然后回到终端博客目录下，输入命令：npm install --save hexo-deployer-git安装插件，安装完成后会出现一个_config.yml的文件，打开它，在文件的最底部进行修改：

注意：冒号后面的空格。

保存完成后输入命令：git config --global user.email "xxx" 和 git config --global user.name "xxx"，设置用户名和邮箱，接下来输入命令hexo -d即可部署到远端，有可能因为网络问题失败，多试几次。成功之后在浏览器输入仓库的地址就可以访问了。芜湖！部署成功！🤪

六、主题

hexo自带一个默认的主题，不喜欢可以自己进行更换，有很多不错的主题，例如Butterfly，Fluid，yilia等等，羊哥演示的就是yilia，我用的是fluid，感觉都很不错。

还是用yilia做演示，git克隆到主题文件夹：git clone https://github.com/litten/hexo-theme-yilia.git theme/yilia 修改_config.yml文件，找到theme，将landscape修改为我们刚刚下载的yilia，保存退出。hexo clean ; hexo g ;hexo -s 可以看到本地的主题已经变了，输入命令：hexo -d推到远端，可能不会马上改变，稍等几分钟，马上就好😉

OK，到这里就结束了，个人博客还有很多其他的玩法，比如自己套个域名，加个评论等等，就靠大家自己去摸索了。

非常感谢程序羊大佬的视频，以及另一位大佬的帮助。
